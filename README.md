## Dependencies

Project Dependencies:

- Create-react-app
- React;
- Sass-rem;
- uuidv1 (to generate unique IDs)

PS: There's no jQuery in this project.

Script to run:

### `npm start`

Script to build:
### `npm run build`

All approved PR sended to the master branch will automatically be deployed and servered with a new link by new.sh
